# 📊 Region Wise Sales Profit Dashboard (Excel)

A dynamic **Sales & Profit Dashboard** built in Microsoft Excel using **Pivot Tables**, **Pivot Charts**, and **Slicers** to analyze regional sales performance. This project provides an interactive way to explore sales data and gain business insights.

---

## 📌 Project Overview

This dashboard helps users analyze sales performance across different regions and years. It provides a quick overview of key business metrics and allows users to filter the data interactively.

---

## ✨ Features

- 📍 Region-wise Sales Analysis
- 📅 Year-wise Sales Comparison
- 📊 Pivot Tables & Pivot Charts
- 🎛️ Interactive Slicers
- 📈 Sales Summary
- 📉 Profit Analysis
- 📌 Dynamic Dashboard

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Formatting
- Dashboard Design

---

## 📊 Dashboard Insights

The dashboard includes:

- Total Sales by Region
- Sales Count
- Average Sales
- Maximum Sales
- Minimum Sales
- Year-wise Sales Distribution (2018–2020)
- Interactive filters using Slicers
